이 프로젝트는 PYBOSSA에서 제공하는 데모 프로젝트로
어떻게 크라우드소싱을 할 수 있는지 볼 수 있습니다.

우리는 매우 간단한 질문을 할 것입니다: **사진 속에 사람이 있나요?**

이 질문에 다음과 같은 답변이 제공됩니다:

  * Yes
  * No
  * I don't know
