How It Works

    • 의뢰인-작업의뢰

      결과물을 정의하고 홈페이지를 통해 작업을 의뢰한다. 원하는 작업을 클라우드 소싱과 전문가의 협업을 통해 적은 비용으로 정확한 데이터를 빠르게 생성한다.

    • 전문가-문제생성

      의뢰된 문제를 작업이 용이한 형태로 재정의하여 작업을 생성한다. 작업 결과를 검증하여 정확한 데이터를 생성한다.

    • 플랫폼

      생성된 작업이 안정적이고 효율적으로 동작하는 환경을 제공한다.

    • 사용자-작업수행

      시관과 장소에 관계없이 작업을 수행한다.
